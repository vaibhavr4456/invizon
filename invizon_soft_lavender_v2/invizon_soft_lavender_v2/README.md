# Invizon Soft Lavender Edition v2

**Features:**

- Amazon-style homepage with product grid and hero banner
- Soft lavender theme with gradient buttons
- Real product images from Amazon, Flipkart, Myntra, Zepto, Meesho, Blinkit
- Header with search filtering
- Profile page with email & saved addresses (add/edit/delete)
- Cart with same-tab redirect: checkout if public, product page if not
- Backend: Node.js + Express + SQLite (persistent)
- JWT authentication
- Demo login: demo@invizon.test / password123

**Run on Replit:**  
1. Upload ZIP or import from GitHub
2. Press **Run** (auto builds frontend + starts backend)
3. Browse products and checkout (redirects handled as described)
