Invizon - Replit-ready Fullstack Monorepo (Persistent SQLite)

Run on Replit:
1. Upload/unzip the project on Replit (Import from ZIP or push to GitHub + import).
2. Press Run (Replit executes `npm start` by default).
3. The app builds the frontend and starts the backend (Express) on the Replit web URL.
4. Demo user: demo@invizon.test / password123

Local run (VS Code):
1. cd backend && npm install && npm run seed && node server.js
2. cd frontend && npm install && npm run dev
